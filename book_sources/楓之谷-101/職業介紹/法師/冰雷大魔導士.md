# 大魔導士 (冰、雷)

![](/images/msn-101/classes-and-jobs/magician/image_1747236396944_492.png)

#### 一位用冰冷及銳利電流制服敵人的法師

大魔導士 (冰、雷) 使用比冰川更寒冷的冰魔法，以及能在敵人之間瞬間流動並無情電擊他們的雷魔法。如果您用雷魔法電擊冰凍的敵人，傷害會增加，讓您能夠造成災難性的一擊。

*   主要屬性：INT (智力)
*   使用武器：長杖、短杖
