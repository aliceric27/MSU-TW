# 大魔導士 (火、毒)

![](/images/msn-101/classes-and-jobs/magician/image_1747236394499_235.png)

#### 一位用強大爆炸和致命劇毒覆蓋戰場的法師

大魔導士 (火、毒) 使用火魔法來燃燒和毒害路徑上所有的敵人。火焰魔法能覆蓋大範圍區域，並對範圍內的敵人持續造成傷害。隨著持續傷害在敵人身上疊加，毀滅性的威力將達到爆炸性的高潮。

*   主要屬性：INT (智力)
*   使用武器：長杖、短杖
